# PavillionDE
Linux's first desktop enviroment (DE) made out of Python.

====== HOW TO  INSTALL ======
1: Go to the releases section and download the zip file (Not the source code)

2: Extract the zip file

3: On the Linux terminal, run the following commands:

1: sudo chmod +x install-pavillion.sh

2: ./install-pavillion.sh

3: Don't worry about essentials.sh as install-pavillion.sh will automatically run it

====== BEFORE YOU  INSTALL ======
The Pavillion Desktop Enviroment is currently in beta, meaning it is not finished yet. The only app you can run is the terminal,
but the rest don't work right now. Also, the mouse cursor is invisible but will soon be visible. You can still run other apps
using the terminal.
